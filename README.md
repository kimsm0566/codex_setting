# Codex Research Setting

This repository stores a reusable snapshot of the working style and project-harness rules that shaped a productive Codex session on a research codebase.

## What Is Included

- `AGENTS.md`
  - a short root policy file for a research repository
- `docs/agent_behavior_rules.md`
  - the practical behavior rules that drove the agent's actions
- `docs/research_harness_summary.md`
  - the research workflow, experiment lifecycle, and evaluation contract summary

## Intended Use

Use this as a starting point when setting up a new research repository where you want:

- plan-first experimental work
- durable documentation under `docs/`
- matched-comparison discipline
- explicit validation and reporting rules
- concise but rigorous agent behavior

## Snapshot Origin

This snapshot was created on `2026-04-03` from a live project session focused on semantic, split, and federated learning research under communication constraints.
