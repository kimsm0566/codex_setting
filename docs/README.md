# Docs Index

This directory is the durable documentation layer for agent-assisted research work.

## Document Map

- `agent_behavior_rules.md`
  - practical behavior rules for the agent
- `research_harness_summary.md`
  - high-level research workflow and promotion logic
- `experiments/README.md`
  - how experiment folders are structured
- `experiments/ACTIVE_PLAN.md`
  - current near-term priorities
- `experiments/RESULTS_LEDGER.md`
  - append-only summary of meaningful runs
- `experiments/_template/`
  - starter files for new experiment attempts
- `evals/EVAL_PROTOCOL.md`
  - evaluation contract and comparison rules

## Working Rules

- Keep durable Markdown documentation here under `docs/`.
- Prefer small, maintainable files over one large instruction dump.
- If code behavior changes in ways that affect evaluation or reproducibility, update the relevant docs in the same task.
